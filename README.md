# Future Tech Portal 🚀

Современный, адаптивный многостраничный веб-портал, посвященный технологиям будущего, подкастам, новостям и аналитическим отчетам. Проект выполнен с использованием модульной структуры стилей (SASS) и компонентного подхода на чистом JavaScript.

---

## 📄 Страницы проекта

Сайт состоит из следующих ключевых страниц:

- `index.html` — Главная страница (Hero-секция, преимущества, общая информация).
- `news.html` — Лента технологических новостей.
- `blog.html` & `blog-details.html` — Блог с возможностью чтения детальных статей.
- `podcasts.html` — Раздел со встроенными или анонсированными аудио-подкастами.
- `resources.html` — Полезные материалы, отчеты и скачиваемые документы.
- `contacts.html` — Форма обратной связи и контактная информация.

---

## 🛠️ Стек технологий

- **Markup:** HTML5 (Семантическая многостраничная верстка)
- **Styles:** SASS/SCSS (Архитектура ITCSS/7-pattern, использование переменных, миксинов и функций)
- **Scripts:** JavaScript ES6+ (Модульная структура, ООП-подход для компонентов)
- **Сборка:** Dart Sass (Компиляция стилей в режиме реального времени)

---

## ⚙️ Архитектура проекта

### Скрипты (`/scripts`)

JS разделен на независимые модули-компоненты для переиспользования кода:

- `Header.js` — Логика шапки и мобильного меню (бургер-кнопка).
- `VideoPlayer.js` — Кастомное управление встроенным видеоплеером.
- `Tabs.js` / `Select.js` — Интерактивные элементы интерфейса (вкладки и выпадающие списки).
- `InputMask.js` / `Feedback-form` — Валидация и маскирование полей ввода в формах.
- `MatchMedia.js` — Адаптивная инициализация скриптов в зависимости от ширины экрана.

### Стили (`/styles`)

Используется компонентный подход. Каждому блоку в HTML соответствует свой `.scss` файл в папке `blocks/` (например, `_hero.scss`, `_video-player.scss`, `_burger-button.scss`). Все файлы импортируются в единый `main.scss`.
